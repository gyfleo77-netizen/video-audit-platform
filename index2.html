 系统功能

本系统基于 MiniCPM-V 4.5 视觉语言模型，提供智能化的视频内容理解与审核能力。

✨ 核心特性

1. 智能场景切分

  •  自动识别视频中的场景切换点
  •  支持两种检测算法：
    • ContentDetector: 适用于快速切换场景（剪辑视频、新闻等）
    • AdaptiveDetector: 适用于渐变场景（电影、纪录片等）
  • ⚖️ 可调节检测灵敏度，平衡检测精度与场景数量

2. 多格式视频支持

  •  自动检测编码格式（H264/AV1/VP9/HEVC）
  •  不兼容格式自动转换
  •  多重读取机制确保最大兼容性

3. 语音识别（ASR）

  • ️ 基于 Faster-Whisper 的高效语音转写
  •  支持多语言自动检测
  •  VAD 过滤提升转写质量

4. 在线视频下载

  •  支持 YouTube、Bilibili 等主流平台
  • ⚡ 基于 yt-dlp 的强大下载能力

 使用流程

步骤 1: 准备视频

  • 方式A：点击「本地视频上传」区域上传文件
  • 方式B：在「在线视频下载」输入 URL 并点击下载

步骤 2: 配置参数

在左侧边栏调整各项参数：

  • ⚙️ 模型配置: 模型路径、量化选项
  •  场景检测: 是否启用、检测算法、灵敏度
  • ️ 抽帧参数: FPS、图像尺寸、时间缩放
  •  提示词: 自定义分析要求
  •  语音识别: 是否启用、模型配置

步骤 3: 开始分析

  • 点击「 开始智能分析」按钮
  • 系统将自动完成：场景检测 → 抽帧 → 语音转写 → 内容分析

步骤 4: 查看结果

  • 切换到「 分析结果」标签页
  • 查看统计数据、完整报告、场景详情
  • 下载 JSON/TXT 格式报告

⚙️ 参数说明

参数 说明 推荐值
场景检测阈值 数值越低越敏感 25-30
抽帧 FPS 每秒采样帧数 1-2
图像最长边 输入模型的图像尺寸 512-768
Beam Size ASR 搜索宽度 5

 最佳实践

✅
快节奏视频（剪辑、Vlog）
• 使用 ContentDetector
• 阈值设置为 20-25
• 抽帧 FPS: 2-3
✅
电影/纪录片
• 使用 AdaptiveDetector
• 阈值设置为 25-30
• 抽帧 FPS: 1-1.5
✅
监控视频
• 禁用场景检测（作为单一场景）
• 抽帧 FPS: 0.5-1

️ 技术架构

  • 视觉理解: MiniCPM-V 4.5
  • 场景检测: PySceneDetect
  • 语音识别: Faster-Whisper
  • 视频处理: Decord / OpenCV / MoviePy
  • 下载工具: yt-dlp
